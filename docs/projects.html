Workbench

Ongoing Projects:

  1. Dialect Coach <https://github.com/nomicflux/dialect-coach> - Chat with AI agents to practice various languages by focusing on local dialects.
  2. Sarf <https://github.com/nomicflux/sarf> - Browser extension to show morphological information for Arabic words.
  3. Entangled Graphs <https://github.com/nomicflux/entangled-graphs> - Visualizations of Quantum circuits.
  4. Danger Mouse <https://github.com/nomicflux/danger-mouse> - A Clojure library for clean handling of errors. Specialized for use in Transducer pipelines.
  5. SCRPG Helper - Helper application for the Sentinal Comics TTRPG, featuring dice statistics, combat tracking, and character creation.
  6. Atomic Reason MCP - MCP server to help AI agents clearly articulate their latent assumptions and provide evidence for them.
  7. Algorithm Visualizations - Visualization of the data structures from Okasaki’s Purely Functional Data Structures, as well as other functional structures.
  8. Servant and Opaleye Tutorial - Setting up a blog with the Haskell web framework Servant and the database library Opaleye.
  9. Hexation - Three-player game based on Domineering, with AIs as players. They’ve been trapped in a cage since birth, learning the game through playing each other - watch how these recluses think and picture the board.
  10. Threals - Threals, a three-player variant on Surreal Numbers and Combinatorial Game Theory. I’ve been working on their theory, though the main holdup is that I cannot seem to find a zero game, so the theory cannot be made into a group.
  11. Guitar Chord Trainer - Guitar chord finder, with working version here.
